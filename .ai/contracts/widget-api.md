# Widget SDK API Contract

Base URL: `/api/v1/public/widget`

## 1. Global Rules

### 1.1 Headers
| Header | Usage | Description |
| :--- | :--- | :--- |
| `X-Channel-Key` | Required for `/config`, `/bootstrap`, `/refresh` | The public key for the app/channel. |
| `X-Visitor-Uid` | Required for `/bootstrap`, `/refresh` | A persistent UUID generated by the client (stored in localStorage). |

### 1.2 Error Schema
All error responses follow this format:
```json
{
  "success": false,
  "message": "Error description",
  "code": "ERROR_CODE", // Optional machine-readable code
  "errors": {
    "field_name": ["Validation error message"]
  }
}
```

### 1.3 Authentication (Messages)
- **401 Unauthorized:** If the `session_token` is expired or invalid.
- **Client Action:** When `401` is received, client **MUST** call `POST /refresh` to get a new token, then retry the request.

## 2. Configuration (Fast Load)

**Endpoint:** `GET /config`
**Purpose:** Fetch UI settings, realtime config, and identity policy. Cached (5 min).

### Request
**Headers:** `X-Channel-Key`, `If-None-Match` (Optional)

### Response (200 OK)
```json
{
  "success": true,
  "data": {
    "widget_config": {
      "widget_color": "#009CE0",
      "welcome_title": "Hi 👋"
    },
    "identity": {
      "mode": "optional", // "optional" | "required"
      "fields": { "name": true, "email": true }
    },
    "realtime": {
      "enabled": true,
      "driver": "reverb",
      "host": "socket.example.com",
      "port": 443,
      "key": "app-key",
      "scheme": "https"
    },
    "meta": {
      "config_version": "hash_string",
      "cache_ttl": 300
    }
  }
}
```

## 3. Bootstrap (Session Start)

**Endpoint:** `POST /bootstrap`
**Purpose:** Authenticate visitor, resolve contact, and start/resume conversation.

### Request
**Headers:** `X-Channel-Key`, `X-Visitor-Uid`
**Body:**
```json
{
  "user": {
    "external_id": "u_123", // Optional
    "email": "jane@example.com", // Optional
    "name": "Jane Doe" // Optional
  },
  "client": {
    "page_url": "https://example.com/pricing",
    "referrer": "google.com"
  }
}
```

### Response (200 OK)
```json
{
  "success": true,
  "data": {
    "session_token": "raw_token",
    "conversation_id": "uuid",
    "contact": { "id": "uuid", "name": "Jane Doe" }
  }
}
```

### Error Responses
**422 Unprocessable Entity (Identity Required):**
```json
{
  "success": false,
  "message": "Identity required",
  "code": "IDENTITY_REQUIRED",
  "errors": {
    "email": ["Email is required to start chat."]
  }
}
```
**403 Forbidden:** If Origin is not allowed.

## 4. Refresh (Token Renewal)

**Endpoint:** `POST /refresh`
**Purpose:** Renew session token using `X-Visitor-Uid`. **Does NOT require the old session token.**

### Request
**Headers:** `X-Channel-Key`, `X-Visitor-Uid`
**Body:**
```json
{
  "conversation_id": "uuid" // Optional: Current conversation ID
}
```

### Response (200 OK)
```json
{
  "success": true,
  "data": {
    "session_token": "new_raw_token",
    "conversation_id": "uuid", // May change if old one was closed
    "contact_id": "uuid"
  }
}
```
**409 Conflict (Bootstrap Required):** If visitor not found. Client MUST call `/bootstrap`.

## 5. Messages

All message endpoints require `Authorization: Bearer <session_token>`.
**Scope:** operations are implicitly scoped to the session's active `conversation_id`.

### List Messages
**Endpoint:** `GET /messages`
**Params:** `?page=1` or `?cursor=...`

#### Response (200 OK)
```json
{
  "success": true,
  "data": {
    "data": [
      {
        "id": "uuid",
        "body": "Hello",
        "type": "visitor", // "visitor" | "user" | "system"
        "visitor_id": "uuid",
        "user_id": "uuid", // Null for visitor
        "sender": { "name": "VisitorName" }, // Or Agent Name/Avatar
        "created_at": "timestamp"
      }
    ],
    "meta": {
      "current_page": 1,
      "last_page": 5,
      "per_page": 50
    }
  }
}
```

### Send Message
**Endpoint:** `POST /messages`
**Body:** `{ "body": "Hello" }`

#### Response (201 Created)
```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "body": "Hello",
    "created_at": "timestamp",
    "type": "visitor",
    "visitor_id": "uuid",
    "sender": { "name": "VisitorName" }
  }
}
```

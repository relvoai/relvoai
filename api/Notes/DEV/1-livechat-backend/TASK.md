# LiveChat Backend Implementation

## Description
Implementation of the CodeCanyon-ready LiveChat + Helpdesk Inbox backend in Laravel 12. This includes full API implementation for Widget (Visitor) and Admin (Agent/Operator), Realtime capabilities via Reverb, and Productivity features.

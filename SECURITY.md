# Security policy

## Reporting a vulnerability

Please do **not** open public GitHub issues for security reports.

Email: security@relvoai.app (placeholder — to be confirmed before public launch).

Include:
- A description of the issue.
- Steps to reproduce.
- The affected version / commit hash.
- Your name (for credit) and any embargo preference.

We will acknowledge within 72 hours and aim to ship a fix within 14 days for
high-severity issues.

## Supported versions

| Version | Support |
|---|---|
| `main` | Active |
| Tagged releases | Latest minor only |

## Coordinated disclosure

We follow standard 90-day coordinated disclosure. Critical issues affecting
the hosted Cloud product will be patched on Cloud first, then in the public
repository at the embargoed date.
